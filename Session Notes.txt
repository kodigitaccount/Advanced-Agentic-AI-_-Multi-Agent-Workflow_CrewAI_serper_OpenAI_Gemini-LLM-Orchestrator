1. **What is the primary characteristic of Agentic AI?**  
   A) It operates without any human intervention.  
   B) It relies solely on a single agent.  
   C) It cannot learn or adapt to its environment.  
   D) It is programmed to only perform simple tasks.

2. **In a Single-Agent System, how many agents are involved?**  
   A) Two or more agents working together.  
   B) Just one autonomous agent.  
   C) It varies based on the environment.  
   D) None; it does not operate.

3. **Which statement best describes a Multi-Agent System?**  
   A) It involves one agent working alone.  
   B) It consists of multiple agents that can communicate and cooperate.  
   C) It operates on a fixed set of tasks without any interaction among agents.  
   D) It is a system that does not adapt to changes in the environment.

4. **Which of the following is an example of Agentic AI?**  
   A) A robot vacuum cleaner that cleans without human help.  
   B) A self-driving car making driving decisions autonomously.  
   C) A clock that keeps time accurately.  
   D) A smartphone running basic applications.

5. **What advantage does a Multi-Agent System have over a Single-Agent System?**  
   A) It is always easier to implement.  
   B) It can handle more complex tasks through collaboration.  
   C) It has fewer components to manage.  
   D) It requires no coordination between agents.
